## 🔧 快速开始

### 1. 安装依赖

```bash
cd /root/.openclaw-claude/workspace/ecommerce-ai-v2
pip install -r requirements.txt
```

### 2. 配置环境变量

```bash
cp .env.example .env
```

编辑 `.env`：

```
DEEPSEEK_API_KEY=sk-xxxxx
SECRET_KEY=your-random-secret-key-here
DATABASE_URL=sqlite:///./ecommerce_ai.db
```

### 3. 运行

```bash
python main.py
```

### 4. 访问

- 首页：http://localhost:8000
- 登录：http://localhost:8000/static/login.html
- 控制台：http://localhost:8000/static/app.html
- 教程：http://localhost:8000/static/tutorials.html
- 定价：http://localhost:8000/static/pricing.html

## 📊 API接口

### 认证相关
- `POST /api/auth/register` - 注册
- `POST /api/auth/login` - 登录
- `GET /api/auth/me` - 获取用户信息

### 功能相关
- `POST /api/select-products` - 智能选品
- `POST /api/analyze-ads` - 投流分析
- `POST /api/pricing/calculate` - 计算定价
- `POST /api/pricing/compare` - 竞品对比
- `POST /api/competitor/analyze` - 竞品分析（新增）
- `POST /api/export` - 数据导出（新增）
- `GET /api/tutorials` - 获取教程列表

### 套餐相关
- `GET /api/plans` - 获取套餐
- `POST /api/purchase` - 购买套餐

## 🚢 部署到Railway

1. 注册Railway账号：https://railway.app
2. 创建新项目
3. 连接GitHub或上传代码
4. 设置环境变量：
   - `DEEPSEEK_API_KEY`
   - `SECRET_KEY`
   - `DATABASE_URL`（Railway自动提供PostgreSQL）
5. 自动部署
